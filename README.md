# Employee Performance Prediction System 🚀

## 📌 Project Overview

This project aims to predict employee performance levels using Machine Learning to support HR decision-making such as promotions, training allocation and performance monitoring.

The model analyses multiple employee attributes including experience, job satisfaction, department, travel frequency and salary hike to estimate performance category.

---

## 📊 Dataset Summary

The dataset contains organisational employee records with features such as:

- Demographics (Age, Gender, Marital Status)
- Work experience metrics
- Job role and department
- Training participation
- Satisfaction scores
- Travel frequency and overtime indicators

Target Variable:
Employee Performance Rating (Multi-class classification)

---

## ⚙️ Machine Learning Workflow

- Data Cleaning and Preprocessing  
- Feature Engineering (Experience Buckets, Encoded Categorical Variables)  
- Handling Class Imbalance using SMOTE  
- Model Training using Random Forest Classifier  
- Hyperparameter Tuning using GridSearchCV  
- Model Evaluation using Accuracy, Recall, ROC-AUC and Precision-Recall Curve
